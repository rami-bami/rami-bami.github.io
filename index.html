Over mij

Mijn naam is Remy.

Ik ben 14 jaar oud en woon in Amsterdam.

Thuis woon ik mijn vader, moeder en 2 zusjes.

Ik speel hockey

Ik zit op het Metis Montessori Lyceum in Amsterdam-Oost. Ik doe het coderclass profiel.

Daar leer je hoe je computers moet coderen.

Ook leer je een aantal programmeertalen. Daar ga ik meer over vertellen in het onderstaande stuk.

Wat kan ik?

Op school doen we coderclass.

Je leer hoe je codeert en verschillende programmeertalen gebruikt.

Dit zijn de talen die ik ken:

Python

Python is een van de eerste programmeertalen die ik heb geleerd.

Met python kan je een programmaatje maken die bijvoorbeeld je naam vraag en die dan in een lijst zet.

HTML

Met html kan je websites maken.

Met css kan je de website mooi maken.

Deze website heb ik ook gemaakt met html.

CSS

Met css kan een website opmaken.

Ik heb css gebruikt om deze website mooi te maken.

Javascript

Javascript wordt vaak gebruikt op het web.

Google gebruikt het om zoek suggesties te geven terwijl je aan het typen bent.

HTML, CSS en javascript worden vaak samen gebruikt om websites te maken.

Blender

Blender is eigenlijk geen programmeertaal.

Maar een programma waar mee je kan 3d modeleren en animeren

Embedded

Met embedded kan je een micro:bit programeren.

Een micro:bit is een robotje die van alles kan.

Bijvoorbeeld een melodietje spelen of lampjes op commando laten branden.

Linux

Linux is een besturings systeem zoals Windows of Mac OS.

Anders dan met Windows ligt de broncode van Linux gewoon vrij.

Dat betekend dat je linux kan aanpassen.

In linux heb ik op de commandlijn gewerkt.

Ik heb geleerd om op de commandlijn bestanden aan te passen.

Blokjesprogrammeren

Blokjesprogrammeren stel eigenlijk niks voor.

Het is met codeblokken een pen of een poppetje te laten bewegen.

De enige uitdaging is op het met zo weinig mogelijk blokken te doen.

Blokjesprogrammeren is een beetje zoals scratch.

Dit zijn ook een aantal badges die ik heb gehaald. Zie meer op de badges pagina.
